A set of bottle icons I am currently creating

Brief

The design and creation of an icon set is the first of two sections to this project. The icon set had be vector based (i.e. created in Adobe Illustrator, Sketch or any other vector-based programme). The final output should be appropriate to the approach you have taken – SVGs, JPGs, PNGs, GIFs, a combination of these, or all four.

With the digital product (icons) in place, I needed to design and build a website to show them off to the best of their ability. The website design should be appropriate to the visual approach taken for your icon set.

Currently the website is in progress.

Sketches

The first thing I did when starting this project was drew out a mindmap, the project brief was quite open so I needed to narrow down my options. I came up witht the first idea of doing bottle icons using this method.

I decided for my idea to stick with the bottle concept but instead of going for something like alcoholic drinks or any drinks I would go for clothing brands. It is something a bit different, I came up with the idea just from doing out a mind map and already sort of having the idea for the bottle concept.

Originally I couldn’t get them to look good because I was just sticking to a flat design but once I decided to add some texture I fell like it was for the better, I was really happy with the way they turned.

I still intent to create some more and I also want to add some form of background to each bottle. I was also thinking of creating something to showcase them all in at once, maybe like a vending machine or example.
